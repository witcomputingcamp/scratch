Objective

On completion of this lab you should be familiar with using Scratch to write simple programs demonstrating flow of control (i.e. Sequence, Selection and Iteration) and basic event handling.

Test your program

  • Click on the green flag to run your program.
  • Does it work as expected?
    • Does your fish say "I'm a fish" and pause for 2 seconds?
    • Does your fish then say "I'm a lovely fish!" and the program stops running?

Saving your work

  • If you are using the Offline Editor on your laptop, save your work to a folder. It is a good idea to create a separate workspace folder in a convenient location and save all your programming there. Give your file the name SomethingFishy1.

  • If you are using the Online Editor, you can download your work to your computer by selecting File from the menu bar followed by the Download to your computer option. Save your work on your G:\ or a USB stick. Give your file the name SomethingFishy1.

Exercises

  • The exercises are typically based on the material we covered in lectures and the previous steps in this lab.

  • For each exercise listed below, create a new Scratch project.

Exercise 1

  • Using a bat sprite, write the following program:

  • Say, "I am a bat!"

  • Repeat the following forever:
    • If the n key is pressed, say "I am nocturnal" for 2 seconds.
    • If the h key is pressed, think "Halloween is my favourite time of the year!".

Exercise 2

  • Choose any sprite you fancy and add any behaviour you wish for the sprite. Remember to include the following flow of control:

    • Sequence
    • Selection (e.g. if)
    • Iteration (e.g. forever, repeat, etc)

Exercise 3

  • Explore the scratch projects on the following website...run them, look inside them at the code, etc.

https://scratch.mit.edu/starter_projects/

  • You will see a lot of different constructs in these projects. We haven't covered some of them, but see if you can figure out what is going on.
